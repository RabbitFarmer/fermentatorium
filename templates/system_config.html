⚠️ Gmail Users: App Password Required

Gmail requires an App Password (not your regular Google password) for this program to send email. To create one: Google Account → Security → 2-Step Verification → App Passwords. Generate a password for "Mail" and use it here. Open App Passwords →

Simpler alternative: Use free ntfy.sh push notifications — zero credentials required. See the Push Notification section above.

⛔ Outlook/Hotmail Personal Accounts: SMTP Not Supported

Microsoft has permanently disabled basic SMTP authentication (username + password) for personal Outlook.com, Hotmail.com, and Live.com accounts. There is no setting in Outlook to re-enable it — this is a Microsoft policy change that affects all personal accounts.

Alternatives that work:

  • ntfy.sh push notifications (recommended) — completely free, no credentials needed. See the Push Notification section above.
  • Gmail with an App Password — create a free Gmail account, enable 2-Step Verification, generate an App Password, and use smtp.gmail.com port 587.
  • Microsoft 365 business/work accounts — an administrator can re-enable SMTP AUTH per-mailbox via the Exchange admin center.
