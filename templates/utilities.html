UTILITIES

These utilities help you manage data, load demo content, and maintain system health. Read each description before running — most operations modify data files. All scripts run from the Fermentatorium install directory.

Demo & Testing
Load Demo Data demo
Populates the system with sample fermentation data so you can explore charts, batch history, and temperature control displays without live hardware. After running, open /chart_plotly/Black to view the demo chart. Running this on a live system will not erase active batches — it writes to designated demo batch IDs.
Script: utils/setup_demo.sh
Verify Demo Data demo
Checks that demo data files were loaded correctly and reports any issues. Safe to run at any time — read-only, no data is modified.
Script: utils/verify_demo_data.py
Data Maintenance
Compact & Archive Logs safe
Splits tilt reading entries from the temperature control log into separate per-batch archive files and rebuilds the main log keeping only the most recent readings per batch. Reduces disk usage over long fermentation runs without losing any data — archived readings are saved in batches/. A backup of the original log is created automatically before any changes are made.
Script: utils/archive_compact_logs.py
Purge Excess Tilt Readings safe
Removes duplicate or over-frequent Tilt readings from batch data files. Useful if an older version of the software logged readings more often than intended (e.g., every 2 minutes instead of every 15 minutes), causing cluttered charts. A backup is created before any records are removed.
Script: utils/purge_excess_tilt_readings.py
Import
Import from Brewer's Friend
Converts a Brewer's Friend JSON export into Fermentatorium's batch format so you can view historical fermentation data in charts and batch history. This utility must be run from the command line — it requires specifying the export file, Tilt color, beer name, and batch name as arguments.
Command-line only. Run from the install directory:
python3 utils/import_brewers_friend.py export.json --color Red --beer-name "My IPA" --batch-name "Batch 7"
Script: utils/import_brewers_friend.py
